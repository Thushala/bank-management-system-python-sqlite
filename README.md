# Bank Management System using Python & SQLite

## Description
This is a console-based Bank Management System developed using Python and SQLite.
The application allows users to perform basic banking operations through a
simple menu-driven interface.

## Features
- Create a new bank account
- Deposit money
- Withdraw money
- Check account balance
- Store data securely using SQLite database
- Perform basic CRUD operations

## Technologies Used
- Python
- SQLite

## How the Project Works
- The program connects to an SQLite database
- Required tables are created automatically if they do not exist
- User inputs are processed through a menu-driven system
- All account details are stored and retrieved from the database

## How to Run the Project
1. Install Python on your system
2. Download or clone this repository
3. Run the main Python file
4. The SQLite database will be created automatically when the program runs
